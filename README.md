# SatuAksi Mobile

SatuAksi Mobile merupakan aplikasi Android yang dikembangkan menggunakan Java di Android Studio.

## Features
- Splash Screen
- Login
- Dashboard
- Text-to-Speech
- Profil Tim
- Informasi Aplikasi

## Tech Stack
- Java
- Android Studio
- XML

## Installation
1. Download file **Satu Aksi.apk** dari repository ini.
2. Aktifkan izin **Install from Unknown Sources** jika diperlukan.
3. Install aplikasi pada perangkat Android.
4. Buka aplikasi dan gunakan.
